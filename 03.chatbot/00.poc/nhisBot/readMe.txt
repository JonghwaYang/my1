기상청 POC 답변 형식

1.링크
----------------------------------------------------------
내용내용내용
<a href="javascript:repeat('URL');"> 감리사업 절차</a>
----------------------------------------------------------
-링크는 답변 그대로 사용


2.재질의 버튼
----------------------------------------------------------
내용내용내용내용
<button> 단계별 주요산출물 확인하기 </button>
----------------------------------------------------------
-재질의 버튼 가공
"<button>" 텍스트가 답변에 있는경우 태그에 class와 onclick 이벤트 추가
한 답변에 여러개의 재질의 버튼있는것까지 고려


3.파일 다운로드
----------------------------------------------------------
내용내용내용내용
<a href="download">다운로드 클릭</a>
----------------------------------------------------------
-파일 다운로드 가공
'href="download"' 텍스트가 답변에 있는 경우 a태그에 경로 이동
최종적으로 DownloadView java Class에 꼳혀서 다운로드 진행
파일은 한가지만 다운받을 수 있도록 코딩


4.이미지
----------------------------------------------------------
내용내욘내용
{intent_01_02_0.jpg,이미지 버튼 내용}
----------------------------------------------------------
-이미지 가공
"{" 와 "}"가 답변에 있는지 확인하고 가공
한 답변에 여러개의 이미지 버튼이 출력되는 것까지 고려
